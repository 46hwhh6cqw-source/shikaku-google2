# NEON NYA SURVIVOR

ルーレットとは無関係の、iPhone Safari / PCブラウザ向けCanvasゲームです。

## 配置場所

```text
neon-nya-survivor/index.html
```

## GitHub Pages URL

このリポジトリでGitHub Pagesが有効なら、以下で開けます。

```text
https://46hwhh6cqw-source.github.io/shikaku-google2/neon-nya-survivor/
```

404になる場合は、GitHubのリポジトリ設定で Pages を有効化してください。

```text
Settings → Pages → Source: Deploy from a branch → Branch: main → Folder: /root → Save
```

## 操作

- iPhone: 指ドラッグで移動
- 右下DASH: 回避
- 攻撃: 自動
- PC: WASD / 矢印 + Space

## iPhoneでアプリっぽく使う

SafariでURLを開いて、共有ボタンから「ホーム画面に追加」。

## 注意

ChatGPTアプリ内のHTMLプレビューや、iPhoneのファイルアプリのQuick LookではJavaScript/Canvasが止まり、黒画面になる場合があります。
GitHub Pagesのような `https://` の普通のWebページとして開くのが前提です。
